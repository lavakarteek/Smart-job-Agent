üß© Step 1B: Use LLM to Find Suitable Job Domains

Once you have structured data:

Call an LLM (GPT-4 or Claude) with a systematic prompt:

import openai
openai.api_key = "YOUR_KEY"

prompt = f"""
You are a career advisor.
Here is a candidate profile:
{profile_json}

Suggest the top 3 job domains/roles that best fit this profile.
Return only a JSON list like ["Role1", "Role2", "Role3"].
"""

response = openai.ChatCompletion.create(
  model="gpt-4",
  messages=[{"role": "system", "content": "You are an expert career advisor."},
            {"role": "user", "content": prompt}]
)
print(response.choices[0].message["content"])


Example Output:
